# Prerequisites
Please install or have installed the following:
* nodejs and npm

# How to use
1. Clone this repo
2. Run `npm install`
3. Run `npm start`
4. You must add your own API key in the src/App.js file to connect to Clarifai.

You can grab Clarifai API key [here](https://www.clarifai.com/)
